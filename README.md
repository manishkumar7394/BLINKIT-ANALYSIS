# 📊 Blinkit Sales Analysis | Python & PostgreSQL

## 📌 Project Overview

This project analyzes Blinkit's sales dataset to uncover meaningful business insights using **Python** for data analysis and visualization, and **PostgreSQL** for querying and extracting insights from the data.

The project demonstrates the complete data analysis workflow—from data cleaning and preprocessing to exploratory data analysis (EDA), business KPI reporting, visualization, and SQL-based analytics.

---

## 🎯 Objectives

- Clean and preprocess raw sales data.
- Perform Exploratory Data Analysis (EDA).
- Analyze sales performance across outlets and product categories.
- Identify key business trends using visualizations.
- Solve business problems using PostgreSQL queries.
- Build a strong foundation in data analytics techniques.
